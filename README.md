Client Side Development - Group 2

Project Structure

    MainWebsite/: The core folder containing all files for the live site.

        images/: The images for the website

        home.html: The main landing page.

        styles.css: All site styling and layout rules.

        javascript.js: Interactive elements and logic.

    Documentation/: Project planning and tracking.

        MinutesLogRAW.txt: A record of our team meetings and progress.

        Wireframes.drawio: Visual blueprints of the site layout.

How to Run

    Clone this repository.

    Navigate to the MainWebsite folder.

    Open home.html in any modern web browser.

Built With

    HTML5 - Page structure

    CSS3 - Custom styling

    JavaScript - Client-side interactivity

    Draw.io - Wireframing and design
